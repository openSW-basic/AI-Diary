
# ✨ 주요 기능

## AI 전화 통화 기반 일기 작성

-   원하는 시간, 요일에 AI 전화를 예약하거나 직접 전화를 걸어 AI와 오늘 하루에 대한 이야기를 나눠요.
-   AI와 전화하며 나눈 내용을 바탕으로 AI가 일기를 작성해줘요.

## AI 감정 분석

-   작성된 일기를 기반으로 AI가 감정을 분석해줘요. 

## AI 답장받기

-   작성된 일기에 대해 AI로부터 따뜻한 답장을 받을 수 있어요.

## 일기 익명 공유(기능 추가 예정)

-   나의 하루를 익명으로 공유해요.
-   일기는 24시간 후 자동으로 비공개 처리되어요.
-   댓글 기능 없이 좋아요 반응만 제공해 따뜻한 공감과 응원의 공간을 마련했어요.

📙 AiRing 사용자 가이드라인 (User Guide)

AiRing은 AI 전화 기반 감성 일기 서비스로, 사용자의 대화를 분석하여 감정과 하루 기록을 일기 형태로 저장해주는 서비스입니다. 이 문서는 앱을 사용하는 일반 유저를 위한 기능 설명과 이용 방법을 제공합니다.

📌 핵심 기능

AI 전화 일기: AI가 정해진 시간에 전화를 걸어 하루를 함께 회고합니다.

자동 일기 생성: AI가 대화 내용을 분석해 일기 요약·감정 분석·키워드를 자동 작성합니다.

감정 캘린더: 하루 감정을 색상으로 시각화하여 월별 감정 흐름을 확인할 수 있습니다.

통화 기록: 날짜별 AI 전화 대화를 다시 읽어볼 수 있습니다.

직접 일기 작성 기능: 사용자가 직접 일기도 기록할 수 있습니다.

🏠 홈 화면 (캘린더 & 요약)
  - 월별 감정과 일기 여부를 색상으로 표시
  - 최근 통화 요약을 보여줌
  - 날짜 선택 시 해당 날짜 일기 또는 통화 기록으로 이동
  - 라이트/다크 테마 지원

⏰ AI 전화 설정 화면
  - 사용자가 원하는 AI 전화 시간을 설정하는 화면입니다.
  - 설정 가능 항목
  - 전화 시간
  - 반복 요일 설정
  - 진동 모드 on/off
  - 다시 전화(부재 시 재연결) 옵션

📔 일기 목록 화면
  - 날짜별 일기 카드 형태로 표시
  - 일기 카드 클릭 시 상세 페이지 이동

📝 일기 상세 화면
  - AI가 생성한 일기 내용 표시
  - 유저가 직접 내용 입력 가능
  - 수정 / 삭제 / 이미지 추가 기능

📞 통화 기록 화면

  - 날짜별 통화 기록 리스트
  - 통화 길이, 감정, 요약 태그 표시
  - 통화 선택 시 대화 전체 내용을 다시 볼 수 있음

🔊 전화 화면
  - AI가 전화를 걸어오는 실제 통화 UI
  - 빨간 버튼: 종료
  - 초록 버튼: 통화 시작
  - 부재 시 "10분 뒤 다시 전화" 가능

🧭 사용자 이용 규칙 (Guidelines)
  - AI 전화는 설정한 시간 기준으로 울립니다 (OS에 따라 몇 초 지연될 수 있음)
  - 일기는 자동 생성되며 사용자가 자유롭게 수정 가능
  - 개인정보 포함된 이미지는 업로드 시 주의
  - 앱 삭제해도 서버 데이터는 유지되며 로그인 시 복원됨
