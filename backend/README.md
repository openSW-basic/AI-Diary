# AiRing Backend

Spring Boot 기반으로 구축된 백엔드 애플리케이션입니다.

## 기술 스택

-   Java 17
-   Spring Boot 3.4.5


## 개발 환경 설정

### 1. 필수 요구사항

-   JDK 17

### 2. 애플리케이션 실행

```bash
# 개발 서버 실행
./gradlew bootRun

# 빌드
./gradlew build

# 테스트
./gradlew test
```
